# Disease_detection_using_medical_imaginarys
Developed a deep learning CNN model to detect diseases from medical images with high diagnostic accuracy.Trained and validated model on labelled datasets; deployed a real-time inference pipeline minimising false positives. 

# 🧠 Disease Detection System using AI/ML  

## 📌 Overview  
This project is a Deep Learning-based system that detects diseases from medical images using Convolutional Neural Networks (CNN).  
It aims to assist in early diagnosis by providing fast and accurate predictions.

---

## 🎯 Features  
- 🧠 CNN-based image classification  
- ⚡ Real-time prediction pipeline  
- 📊 High accuracy model  
- 🖼️ Image preprocessing and augmentation  
- 🔍 Easy-to-use interface (optional if UI added)  

---

## 🛠️ Tech Stack  

- **Language:** Python  
- **Libraries:** TensorFlow, Keras, OpenCV, NumPy, Pandas  
- **Visualization:** Matplotlib  
- **Tools:** Jupyter Notebook / VS Code
